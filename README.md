# c-
----Variables y tipos de datos:

int, float, double, char, bool.

Declaración e inicialización: int x = 10;.

------Estructuras de control:

Condicionales: if, else if, else, switch.

Bucles: for, while, do-while.

------Funciones:

Declaración: tipo nombreFuncion(parametros).

Llamada: nombreFuncion(argumentos).

------Arreglos y cadenas:

Arreglos: int vec[5] = {1, 2, 3, 4, 5};.

Cadenas: string texto = "Hola";.

------Punteros y memoria dinámica:

Declaración: int* ptr = &x;.

Memoria dinámica: new y delete.

Estructuras y clases:

Estructuras: struct Nombre { ... };.

Clases: class Nombre { ... };.
